# Maîtres Fleuristes Elhadji, Stephan & Arole 

## 1. Contexte du projet

Ce projet a été réalisé dans le cadre du cours **UA1 – Communication en Entreprise**.  
Il consiste à concevoir le site web vitrine d’une boutique fictive de fleurs nommée **Maître Fleuriste Elhadji, Stephan & Arole **.

L’objectif principal était de créer une interface moderne, claire et responsive, inspirée d’un site e-commerce existant, tout en respectant les bonnes pratiques du développement web (séparation des fichiers, lisibilité du code, interactivité et communication).

---

## 2. Objectifs du projet

Les objectifs de ce projet sont les suivants :

- Concevoir une page web esthétique et professionnelle
- Structurer correctement un projet web (HTML / CSS / JavaScript)
- Mettre en place une boutique avec :
  - une liste de produits
  - un système de panier 
- Utiliser Git et GitHub pour la gestion de versions

---

## 3. Méthodologie et approche utilisée

Le projet a été réalisé en plusieurs étapes :

1. **Analyse d’un modèle existant** (inspiration visuelle et structurelle)
2. **Création de la structure HTML**
   - balisage sémantique
   - sections claires (en-tête, produits, filtres, pied de page)
3. **Mise en forme avec CSS**
   - grille responsive
   - animations hover
4. **Ajout d’interactivité avec JavaScript**
   - gestion du panier
   - filtres dynamiques (prix et catégories)
5. **Gestion du projet avec Git et GitHub**
   - commits réguliers
   - dépôt distant sur GitHub

---

## 4. Résultats et visualisations

Le résultat final est un site web fonctionnel comprenant :

- Une page d’accueil claire et professionnelle
- Une section de produits avec images, prix et boutons d’ajout au panier
- Un panier avec compteur dynamique
- Une animation au survol des produits
